## Versioning policy

ai-memory follows [Semantic Versioning](https://semver.org/):

- **Major** (`x.0.0`) — breaking API changes
- **Minor** (`0.x.0`) — new features, additive schema migrations
- **Patch** (`0.0.x`) — bug fixes, security patches, no schema breaks
