Persistent memory, knowledge graph, and governance for AI agents.

ai-memory is the autonomous-tier substrate for AI Non-Human Identity (NHI) agents. Typed memory. A knowledge graph with temporal validity. Signed reflections. One Rust binary, Apache-2.0.

Install What is it?
Install

Three install paths. Pick one.

A single static Rust binary, no daemons required to get started. Full install matrix (Windows, Docker, .deb, .rpm, source) in the install reference.

macOS / Linux
Homebrew
brew install alphaonedev/tap/ai-memory
Fedora / RHEL
COPR
sudo dnf copr enable alpha-one-ai/ai-memory
sudo dnf install ai-memory
Any platform
cargo
cargo install ai-memory

Verify:

ai-memory --version
ai-memory 0.7.0
